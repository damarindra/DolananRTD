Dolanan documentation
=====================

Welcome to the official documentation of Dolanan, a MonoGame Framework extension
contains basic function for creating game, such as sprite, camera, collision etc.
Dolanan will focus on usability and simplicity for creating 2D game.

Dependency
----------

* MonoGame Portable 3.7.1.189 or higher (https://monogame.net/)

* Sigil 5.0.0 or higher (https://github.com/kevin-montrose/Sigil)

Related Tools
-------------

* Aseprite (https://www.aseprite.org/)
